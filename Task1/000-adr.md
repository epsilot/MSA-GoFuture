### Нефункциональные требования

| Код | Требование                                                                                         |
|-----|----------------------------------------------------------------------------------------------------|
| R1  | Отказоустойчивость 99.95%                                                                          |
| P1  | Обработка более 500 000 конкурентных поездок                                                       |
| P2  | Поддержка автоматического масштабирования на основе текущей нагрузки в разных регионах присутствия |
| P3  | Время сборки не должно превышать 15 минут для основных компонент системы                           |
| S1  | Полное покрытие критических метрик и генерация соответствующих алёртов                             |
| U1  | Возможность разворачивать платформу с нуля в течение 2х недель                                     |
| +R1 | Централизованное IAM                                                                               |
| +R2 | Миграция данных должна быть выполнена без даунтайма                                                |
| +R3 | Соблюдение локальных регуляций                                                                     |
